## API overview

This documentation covers the **public Lightrail API surface** used for day-to-day integrations.
It focuses on external workflows such as authenticating clients, working with accounts, creating orders and payouts, and retrieving receipts and transaction history.
Use this version when you want a clean, stable view of endpoints intended for standard API consumers.
