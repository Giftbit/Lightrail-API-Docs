## API overview

This documentation covers the **full Lightrail API surface**, including the public integration endpoints plus advanced operational APIs.
Alongside orders, payouts, receipts, and transactions, this version is intended to include deeper account configuration and administrative capabilities used by platform and internal teams.
Use this version when you need end-to-end visibility across both integration and management endpoints.
